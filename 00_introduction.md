# Linuxに関するイントロダクション
## 環境: Linux

1. 以下のコマンドを実行し、動きを確認しましょう。  
  * ls (オプション -l, -a など) 
  * touch
  * mkdir
  * cp
  * mv
  * rm
  * exit

2. vi, emacsのいずれかの使い方を覚えましょう。現時点では与えられた任意のテキストを入力して保存できれば合格です。  

  (a) viの場合  
    _必ず確認しておくべきコマンド_
    * h, j, k, l
    * i, I
    * a, A
    * r
    * d,dd
    * u
    * o, O
    * p, P
    * .
    * /
    * :w
    * :ls
    * :b  

  _お奨めコマンド_  
    * f, F  
    * t, T  

  (b) emacsの場合  
    分からないので自力で調べてください。
